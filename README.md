##  Core Projects Spotlight

###  TubeChat — RAG-based Chatbot
An advanced Retrieval-Augmented Generation system that indexes multi-hour YouTube transcripts. Built with LangChain and LangGraph, it uses local embeddings (`all-MiniLM-L6-v2`), FAISS vector storage, CrossEncoder reranking, and Groq-hosted LLM inference to deliver fast, context-aware responses with timestamp grounding.

---

###  Sherlock Text Generator — LSTM-based Next Word Predictor
A deep learning text generation model built with stacked LSTMs trained on classical literature. Supports temperature scaling and top-k sampling to control randomness and improve output quality.

---

###  Customer Category Classification
An ML pipeline for customer segmentation using supervised and unsupervised learning. Includes preprocessing, feature engineering, clustering, and predictive modeling to identify behavioral patterns.
